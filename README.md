# Welcome to Dream5!

## What is Dream5?

Dream5 is a mobile-only web application that allows users to bet on the teams to get scores.

## Tech Stack

- React Router v7
- Tailwind CSS
- Shadcn UI
- Vite
- Prisma as the ORM
- Supabase as the database
- Vercel for hosting and deployment

Built with ❤️ using React Router.
